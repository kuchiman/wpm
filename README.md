# wpm
Windosws package manager

Пакетный менеджер для Windows. Попытка создать систему управления ПО для небольшой организации. На данный момент представляет из себя консольную 
программу способную устанавливать, обновлять и удалять ПО из репозиториев. Может работать с несколькими репозиториями.

Заготовки для пакетов существуют в виде отдельного репозитория [wpm-pkg](https://github.com/kuchiman/wpm-pkg)
